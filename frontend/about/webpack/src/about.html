Who created?
Hi, my name is Sasha and I am the creator of GitHarbor,
which is based on the stack - Vue-GWT / Javalin / Dagger2
Why I created GitHarbor
As a developer, I constantly need to read other people's sources, so I always have a lot of tabs open on GitHub. Each new tree file opens in a new tab, between tabs you need to constantly switch, look for the ones you need and close not relevant ones. And I would like simplicity and ease, as in your favorite development environment. So the first prototype of GitHarbor was born - a simple lazy tree and tabs, and then it started, screwed Java navigation, a file search appeared. There was an interesting service that could make life easier not only for me, but also for other developers.
What is the GitHarbor about?
GitHarbor reminds Intellij IDEA, created specifically for this project. Using this service. With the help of GitHarbor, you can quickly navigate the repository using the tree and find the files you need, search for the code in the repository and subsequent opening, open several files at once in one tab in the log.
